Hexo博客管理后台

使用GitHub Personal Access Token进行认证

1

生成GitHub Token

访问 GitHub Token设置

点击"Generate new token"按钮

选择 repo 权限（全选），然后生成token

2

复制Token

复制生成的token（以ghp_或github_pat_开头）

3

开始使用

点击下方按钮保存token并进入管理后台

重要提示：
  • Token只会保存在你的浏览器本地，不会发送到任何服务器
  • 请妥善保管你的token，不要分享给他人
  • Token需要repo权限才能读写仓库
  • 如果token过期或失效，重新生成一个新的即可
